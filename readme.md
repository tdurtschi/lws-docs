## Quick build

`pushd mkdocs; mkdocs build -d ../docs; popd;`

## Local setup 

To install a virtual env:
```sh
python3 -m venv .venv
source .venv/bin/activate
```

Install mkdocs:
```sh
pip install mkdocs
```

Also see https://www.mkdocs.org/getting-started/
