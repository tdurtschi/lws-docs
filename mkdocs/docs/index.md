# Lake Wingra Software

![Lake Wingra](img/lakewingra.jpg)

Lake Wingra Software was an independent software consulting company based in Madison, Wisconsin, operating from 2022 to 2026. We partnered with teams across healthcare, manufacturing, retail, and agriculture to deliver high-quality software through Test-Driven Development, refactoring, and DevOps practices.

Lake Wingra Software is not currently taking on new work. Thank you to everyone we had the pleasure of working with!
